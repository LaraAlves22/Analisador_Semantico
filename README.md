# 🧩 Analisador Semântico – Compiladores

Este projeto implementa a **análise semântica** de um compilador, sendo responsável por verificar a **correção lógica e de significado** de um programa após a análise sintática.

A análise semântica garante que o código fonte, além de sintaticamente correto, também respeite regras como **tipagem, escopo e declarações válidas**.

---

## 🎯 Objetivo

Implementar um módulo capaz de:

- Validar regras semânticas da linguagem
- Detectar erros relacionados a tipos, escopo e uso de identificadores
- Garantir consistência nas operações do programa

---

## 🧠 Conceitos Aplicados

- 🧾 Tabela de símbolos  
- 🔤 Tipagem (type checking)  
- 📍 Escopo de variáveis  
- 🔁 Verificação de declarações  
- ⚠️ Detecção de erros semânticos  
- 🌳 Travessia de árvore sintática (AST)

---

## 🚀 Funcionalidades

- ✅ Verificação de variáveis não declaradas  
- ✅ Detecção de redeclaração de identificadores  
- ✅ Checagem de tipos em expressões  
- ✅ Validação de atribuições   
- ✅ Geração de mensagens de erro semântico
