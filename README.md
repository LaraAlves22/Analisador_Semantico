# 🧩 Projeto – Compiladores

Este projeto foi desenvolvido como parte da disciplina de **Compiladores**, com o objetivo de implementar etapas fundamentais de um compilador, como **análise léxica e sintática**.

---

## 🎯 Objetivo

Construir um sistema capaz de processar uma linguagem de entrada e realizar sua análise, identificando erros e/ou gerando estruturas intermediárias.

---

## 🧠 Conceitos Aplicados

- 🔤 Análise Léxica (Tokenização)
- 📏 Análise Sintática (Parsing)
- 🧩 Gramáticas Livres de Contexto (GLC)
- ⚙️ Autômatos (AFD/AP)
- ❗ Tratamento de erros
- 📜 Processamento de linguagem formal

---

## 🚀 Funcionalidades

- ✅ Leitura de código fonte de entrada  
- ✅ Identificação de tokens  
- ✅ Verificação da estrutura sintática  
- ✅ Detecção de erros léxicos e sintáticos  
- ✅ Geração de árvore sintática
